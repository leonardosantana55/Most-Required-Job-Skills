APLICATIVO "mrjskills.com" - Most Required Job Skills


FUNCIONALIDADE
Pesquisa descrições de vagas de emprego e retorna as palavras mais mencionadas, gerando insights de quais habilidades e ferramentas são mais requisitadas para o mercado de trabalho.


ARQUITETURA

WEB APP
Recebe query do usuário e envia request para a api
Recebe resposta da api e apresenta informações


API
Recebe a solicitações e envia para o backend
Recebe a resposta do backend e a encaminha para o aplicativo


BACK END
Recebe solicitações por query
Consulta banco de dados
Se a query já existe, retorna a consulta
Se a query não existe, começa um JOB e envia um aviso.


BACK END JOB
Coleta dados realizando webscrapping
Cria um dataset com o resultado
Consulta no banco de dados a query existente
Mescla os dados do dataset do scrapping com o do banco de dados
Atualiza o banco de dados
Repete o processo no dia seguinte

BANCO DE DADOS POSTGRES
Esquema transacional
Formato: cada linha um token, cada coluna uma query
